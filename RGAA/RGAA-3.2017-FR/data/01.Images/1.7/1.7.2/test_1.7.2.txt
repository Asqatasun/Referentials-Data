Chaque bouton de type image (balise input avec l'attribut type="image") porteur d'information, ayant une description détaillée, vérifie-t-il ces conditions ? La description détaillée dans la page et signalée dans l'attribut alt est pertinente. La description détaillée via un lien adjacent est pertinente. Le passage de texte référencé via la propriété aria-describedby est pertinent.
