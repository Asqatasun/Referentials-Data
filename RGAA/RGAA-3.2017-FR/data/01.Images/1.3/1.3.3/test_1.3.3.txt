Chaque bouton de type image (balise input avec l'attribut type="image"), ayant un attribut alt, vérifie-t-il ces conditions (hors cas particuliers) ? Le contenu de l'attribut alt est pertinent. S'il est présent, le contenu de l'attribut title est identique au contenu de l'attribut alt. S'il est présent, le contenu de la propriété aria-label est identique au contenu de l'attribut alt. S'il est présent, le contenu du passage de texte lié via la propriété aria-labelledby est identique au contenu de l'attribut alt.
