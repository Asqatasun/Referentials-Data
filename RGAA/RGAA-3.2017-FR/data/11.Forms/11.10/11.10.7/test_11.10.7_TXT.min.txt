Test 11.10.7 : Pour chaque formulaire, chaque champ obligatoire vérifie-t-il une de ces conditions ? Le type de données et/ou de format est indiqué, si nécessaire, dans l'étiquette du champ. Le type de données et/ou de format est indiqué, si nécessaire, par un passage de texte avant le champ de formulaire. Le type de données et/ou de format est indiqué, si nécessaire, par un passage de texte lié par la propriété ARIA aria-describedby.
